# 🚀 Quick Start Guide - Vizag Tours & Travels

## 🎯 Get Started Immediately (No Setup Required!)

### Option 1: Use the Standalone HTML Version

1. **Open the application** - Simply open `index.html` in your web browser
2. **No installation needed** - It runs entirely in your browser
3. **Data is saved locally** - All bookings are stored in your browser

**File location:** `c:\Users\sreenath\Desktop\Travels Application\index.html`

**Steps:**
- Right-click on `index.html`
- Select "Open with" → Choose your browser (Chrome, Firefox, Edge, etc.)
- The application will load immediately!

---

## 📋 Features in Standalone Version

### For Users (Booking Page)
✅ Browse available cabs with details
✅ Fill booking form with personal information
✅ Select pickup and drop locations
✅ Choose date and number of passengers
✅ Select cab type and view pricing
✅ Get instant booking confirmation with Booking ID

### For Admin (Admin Dashboard)
✅ View all bookings in a detailed table
✅ See real-time statistics (total, pending, confirmed, completed)
✅ Change booking status (Pending → Confirmed → Completed → Cancelled)
✅ Delete bookings
✅ All data persists in browser

---

## 💾 Data Storage

All booking data is stored in your browser's **localStorage**. This means:
- Data persists even after closing the browser
- Data is stored locally on your computer
- Refreshing the page keeps all bookings
- Each browser/device has separate data

## 🔧 Full-Stack Version (Optional Setup)

If you want the complete Node.js backend + React frontend version:

### Prerequisites
- Install **Node.js** from https://nodejs.org/ (v14 or higher)

### Setup Backend

```bash
cd "c:\Users\sreenath\Desktop\Travels Application\backend"
npm install
npm start
```

Backend will run on: **http://localhost:5000**

### Setup Frontend (in new terminal)

```bash
cd "c:\Users\sreenath\Desktop\Travels Application\frontend"
npm install
npm start
```

Frontend will open at: **http://localhost:3000**

## 🆘 Troubleshooting

### Can't open index.html?
- Right-click on the file and select "Open with"
- Choose your preferred browser

### Data not saving?
- Make sure browser allows localStorage
- Check privacy settings in your browser

### Can't run npm commands?
- Install Node.js from https://nodejs.org/
- Restart your terminal after installation
