## 📊 **Booking Status Flow**

```
Pending ──→ Confirmed ──→ Completed
    ↓
 Cancelled
```

- **Pending:** Initial booking state, awaiting admin confirmation
- **Confirmed:** Admin has verified and confirmed the booking
- **Completed:** Trip has been successfully completed
- **Cancelled:** Booking has been cancelled
